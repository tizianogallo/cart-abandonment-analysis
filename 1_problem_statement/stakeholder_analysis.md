## 4. 🔍 Stakeholder Needs & Expectations

### 👔 E-Commerce Director
- **Needs:** Clear business case with quantified revenue impact and a prioritised roadmap
- **Expects:** Executive-level summary with ROI projections
- **Risk if not engaged:** Project lacks sponsorship and recommendations go unimplemented

### 📣 Marketing Manager
- **Needs:** Data on when and why users abandon, to inform email recovery campaigns
- **Expects:** Segmented abandonment data by category and time of day
- **Risk if not engaged:** Recovery campaigns are generic and ineffective

### 🎨 UX Designer
- **Needs:** Specific friction points in the checkout flow backed by data
- **Expects:** AS-IS process map highlighting drop-off points with supporting analysis
- **Risk if not engaged:** Redesign misses the highest-impact pain points

### 💻 Development Lead
- **Needs:** Technically feasible requirements with clear acceptance criteria
- **Expects:** User stories written in a format ready for sprint planning
- **Risk if not engaged:** Recommendations are impractical to implement

### 💰 Finance Analyst
- **Needs:** Conservative, well-evidenced ROI estimates
- **Expects:** Clear assumptions behind revenue recovery projections
- **Risk if not engaged:** Business case lacks credibility for budget approval
